# Button
Button variants
```
<button class="btn">neutral</button> 
<button class="btn primary">primary</button> 
<button class="btn secondary">secondary</button> 
<button class="btn accent">accent</button> 
<button class="btn ghost">ghost</button> 
<button class="btn link">link</button>
```
Outline button
```
<button class="btn outline">neutral</button>
```
Disabled button
```
<button class="btn" disabled="disabled">neutral</button> 
<button class="btn disabled" tabindex="-1" role="button" aria-disabled="true">visually disabled</button>
```
Active by default
```
<button class="btn active" role="button" aria-pressed="true">neutral</button>
```

```
<button class="btn lg">Large</button> 
<button class="btn">Normal</button>

<button class="btn xs">Tiny</button>
```
