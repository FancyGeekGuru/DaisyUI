Styled (and unstyled) UI Components for Tailwind CSS users


# Install  

```
npm i daisyui
```

## Add plugin and preset to `tailwind.config.js`
```
module.exports = {
  plugins: [
    require('daisyui'),
  ],
  presets: [
    require('daisyui/preset')
  ],
}

```

# Or use a CDN
```
<link rel="stylesheet" href="https://unpkg.com/daisyui@latest/dist/styled.min.css" />

```
