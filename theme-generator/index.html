daisyUI Theme Generator

You can add your custom themes to tailwind.config.js file in daisyui > themes array. In this page, you can pick required color values and see how the components will look like with them.

You can also define optional colors to have more control on the color values (for example: the color of button when it's focused of the color of the text on a button)

Check out colors page to see all the color names you can use.

Check out themes page to see all the CSS variables you can use to customize the design decision (example: border-radius, animations, etc.)
